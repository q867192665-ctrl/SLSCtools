## 三、打包步骤

### 方式一：一键构建（推荐）

```powershell
cd "e:\程序\chinantool"
python _build_all.py
```

自动完成：清理 → PyInstaller打包后端 → PyInstaller打包启动器 → 整理文件 → Inno Setup编译

**注意**：`_build_all.py` 中 Inno Setup 步骤因路径含中文可能静默失败，需手动补一步：
```powershell
& "C:\Program Files (x86)\Inno Setup 6\ISCC.exe" "e:\程序\chinantool\setup.iss"
```

### 方式二：手动逐步执行

#### 步骤1：清理旧构建
```powershell
Remove-Item -Recurse -Force "e:\程序\chinantool\build","e:\程序\chinantool\dist" -ErrorAction SilentlyContinue
```

#### 步骤2：打包后端（chinantool_backend.exe）
```powershell
cd "e:\程序\chinantool"
pyinstaller --noconfirm --onefile --noconsole `
    --name "chinantool_backend" `
    --icon "logo.ico" `
    --hidden-import "uvicorn.logging" `
    --hidden-import "uvicorn.loops" `
    --hidden-import "uvicorn.loops.auto" `
    --hidden-import "uvicorn.protocols" `
    --hidden-import "uvicorn.protocols.http" `
    --hidden-import "uvicorn.protocols.http.auto" `
    --hidden-import "uvicorn.protocols.websockets" `
    --hidden-import "uvicorn.protocols.websockets.auto" `
    --hidden-import "uvicorn.lifespan" `
    --hidden-import "uvicorn.lifespan.on" `
    --hidden-import "yaml" `
    --hidden-import "routeros_api" `
    --hidden-import "routeros_api.api_structure" `
    --hidden-import "routeros_api.resource" `
    --hidden-import "routeros_api.socket_api" `
    --hidden-import "routeros_api.communication" `
    --hidden-import "routeros_api.api_communication" `
    --hidden-import "websockets" `
    --hidden-import "websockets.legacy" `
    --hidden-import "websockets.legacy.server" `
    --hidden-import "psutil" `
    --hidden-import "PIL" `
    main.py
```

#### 步骤3：打包启动器（ChinanTool.exe）
```powershell
pyinstaller --noconfirm --onefile --noconsole `
    --name "ChinanTool" `
    --icon "logo.ico" `
    --hidden-import "yaml" `
    launcher.py
```

#### 步骤4：整理文件
```powershell
$setupDir = "e:\程序\chinantool\dist\setup_files"
New-Item -ItemType Directory -Force -Path $setupDir
Copy-Item "e:\程序\chinantool\dist\chinantool_backend.exe" $setupDir
Copy-Item "e:\程序\chinantool\dist\ChinanTool.exe" $setupDir
Copy-Item "e:\程序\chinantool\config.yaml" $setupDir
Copy-Item "e:\程序\chinantool\logo.ico" $setupDir
Copy-Item "e:\程序\chinantool\Logo.jpg" $setupDir
Copy-Item -Recurse -Force "e:\程序\chinantool\static" "$setupDir\static"
```

#### 步骤5：Inno Setup 编译安装包
```powershell
& "C:\Program Files (x86)\Inno Setup 6\ISCC.exe" "e:\程序\chinantool\setup.iss"
```

## 五、关键代码注意事项

### 1. `get_base_dir()` — 所有文件必须使用
PyInstaller 打包后 `__file__` 指向临时目录，必须用此函数获取exe所在目录：
```python
def get_base_dir():
    if getattr(sys, 'frozen', False):
        return os.path.dirname(sys.executable)
    return os.path.dirname(os.path.abspath(__file__))
```
**涉及文件**：`main.py`、`launcher.py`、`api_server.py`、`websocket_server.py`

### 2. `NullWriter` — --noconsole 模式必须
`--noconsole` 时 `sys.stdout`/`sys.stderr` 为 None，uvicorn 调用 `.isatty()` 会崩溃：
```python
if getattr(sys, 'frozen', False):
    class NullWriter:
        def write(self, *args): pass
        def flush(self): pass
        def isatty(self): return False
    if sys.stdout is None:
        sys.stdout = NullWriter()
    if sys.stderr is None:
        sys.stderr = NullWriter()
```
**涉及文件**：`main.py` 的 `main()` 函数开头

### 3. 日志写入 LOCALAPPDATA — 避免权限错误
安装到 Program Files 后无法写日志，必须写到用户目录：
```python
if getattr(sys, 'frozen', False):
    local_app = os.environ.get('LOCALAPPDATA', '')
    if local_app:
        log_dir = os.path.join(local_app, 'ChinanTool')
    else:
        log_dir = get_base_dir()
    os.makedirs(log_dir, exist_ok=True)
```
**日志路径**：`%LOCALAPPDATA%\ChinanTool\chinantool.log`、`%LOCALAPPDATA%\ChinanTool\launcher.log`

### 4. PyInstaller 参数
- `--noconsole`：不显示CMD黑窗口
- `--onefile`：打包为单个exe
- `--noconfirm`：覆盖已有输出
- `--hidden-import`：uvicorn/websockets/routeros_api 等动态导入需要手动声明

### 5. 设备发现逻辑
- 后端每10秒自动发送MNDP发现报文（`start_auto_discover(interval=10)`）
- 收到MNDP报文立即显示设备（监听线程更新 `discovered_devices`）
- 点刷新：清空设备列表 → 发发现包 → 等3秒 → 返回在线设备（`/api/refresh`）
- 不点刷新不删设备

## 七、常见问题

| 问题 | 原因 | 解决 |
|------|------|------|
| `PermissionError` 写日志 | 安装到 Program Files 无写权限 | 日志写 LOCALAPPDATA |
| `AttributeError: 'NoneType'.isatty()` | --noconsole 下 stderr 为 None | NullWriter 替换 |
| `{"detail":"Frontend not found"}` | __file__ 在 PyInstaller 临时目录 | 用 get_base_dir() |
| 离线设备刷新后不删除 | 旧逻辑只删过期不验证在线 | 刷新时清空列表+重新发现 |
| Inno Setup 静默失败 | _build_all.py 路径含中文 | 手动运行 ISCC.exe |
